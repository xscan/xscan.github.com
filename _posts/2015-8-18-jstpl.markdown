#1.前言

什么是模板引擎，说的简单点，就是一个字符串中有几个变量待定。比如：

		var tpl = 'Hei, my name is <%name%>, and I\'m <%age%> years old.';

		var data = {
		    "name": "Barret Lee",
		    "age": "20"
		};
		 
		var result = tpl.replace(/<%([^%>]+)?%>/g, function(s0, s1){
		    return data[s1];
		})
		document.write(result)

以上就只有三个操作：
		
		 1.模板
		 2.数据
		 3.正则替换
		 4.输出到浏览器
